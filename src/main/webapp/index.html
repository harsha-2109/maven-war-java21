☕ Maven WAR — Java 21 Demo

This application showcases Java 21 features in a classic Maven WAR deployment:

  • Records — Product and ApiResponse
  • Sealed interfaces — ApiResponse<T>
  • Pattern matching in switch — used in servlets & service layer
  • Text blocks & String.formatted() — throughout the codebase

API Endpoints

MethodURLDescription
GET/api/healthHealth check
GET/api/productsList all products
GET/api/products/{id}Get product by ID
POST/api/productsCreate a new product
PUT/api/products/{id}Update a product
DELETE/api/products/{id}Delete a product

Build & Run

mvn clean package
# Deploy the generated WAR to Tomcat 10+, WildFly 27+, or GlassFish 7+
